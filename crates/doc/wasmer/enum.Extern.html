Enum wasmer::Extern

source · 
pub enum Extern {
    Function(Function),
    Global(Global),
    Table(Table),
    Memory(Memory),
}
Expand description

An Extern is the runtime representation of an entity that can be imported or exported.

Spec: https://webassembly.github.io/spec/core/exec/runtime.html#external-values
